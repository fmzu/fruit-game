# Fruit Game — CLIで動くフルーツキャッチゲーム

## コンセプト

ターミナル上で遊べるフルーツキャッチゲーム。上から落ちてくるフルーツをプレイヤーが左右に移動してキャッチし、スコアを稼ぐ。スコアが上がると高得点フルーツが出現するようになる。3回取り逃すとゲームオーバー。

## 使い方

```bash
bun run dev
```

## 仕様

### ゲームルール

- プレイヤーはフィールド最下段を左右に移動し、落下するフルーツをキャッチする
- フルーツをキャッチするとスコアが加算される
- フルーツがフィールド外に落下するとミス（ライフが1減る）
- ライフは3（ミス3回でゲームオーバー）
- キャッチ判定: プレイヤー（幅4）とフルーツ（幅2）の矩形が重なっているか

### フルーツの種類

| フルーツ | 絵文字 | 得点 | 出現条件（スコア） |
|----------|--------|------|---------------------|
| りんご   | 🍎     | 5    | 0以上（最初から）   |
| オレンジ | 🍊     | 10   | 20以上              |
| ぶどう   | 🍇     | 15   | 50以上              |
| メロン   | 🍈     | 25   | 100以上             |

- スコアが条件を満たすと、そのフルーツが出現候補に追加される
- 出現候補の中からランダムに選ばれる（均等確率）

### フルーツの出現・落下

- 80フレームごと（60fpsで約1.3秒間隔）にフィールド上端（y=0）のランダムなx座標に1つ出現
- 8フレームごと（60fpsで約0.13秒間隔）にy座標が1ずつ増加して落下

### プレイヤーの移動

- 2フレームごとに1ステップ移動（60fpsで毎秒30ステップ）
- フィールド端で停止（0 <= x <= fieldWidth - playerWidth）
- キー押下中は継続移動（keysHeldフラグで管理）
- キーリリース検出: 最後のキー入力から150ms後にリリースとみなす

### フィールド

- 幅40文字 x 高さ20行
- 罫線（┌─┐│└─┘）で囲まれた枠内がプレイエリア

### 操作方法

| キー | アクション |
|------|------------|
| 左矢印 (←) | プレイヤーを左に移動 |
| 右矢印 (→) | プレイヤーを右に移動 |
| Enter / Space | ゲーム開始（タイトル画面） |
| R | リスタート（ゲームオーバー画面） |
| Ctrl+C | 終了 |

### 画面構成

**タイトル画面**
- 中央揃えで「FRUIT GAME」タイトル、操作説明、開始案内を表示
- Enter/Spaceで開始

**ゲーム画面**
- ヘッダー: スコアとライフ（♥/♡）
- フィールド: 罫線枠内にフルーツ（絵文字）とプレイヤー（`[==]`）を描画

**ゲームオーバー画面**
- 中央揃えで「GAME OVER」、最終スコア、リスタート案内を表示
- Rキーでリスタート

## 技術構成

- **言語**: TypeScript
- **ランタイム**: Bun
- **TUIライブラリ**: @opentui/core（ターミナル描画、キー入力、alternate-screen）
- **フレームレート**: 60fps（setIntervalベースのゲームループ）
- **アーキテクチャ**: 一関数一ファイル、純粋関数ベースの状態管理（イミュータブル更新）
- **テスト**: bun test

### ファイル構成

```
src/
  index.ts                  # エントリポイント（レンダラー初期化、ゲームループ、キー入力）
  types.ts                  # 型定義（GameState, Fruit, Player, Action等）
  create-initial-state.ts   # 初期状態の生成
  update-state.ts           # ゲームロジック統合（tick/入力に応じた状態遷移）
  move-player.ts            # プレイヤー移動
  move-fruits.ts            # フルーツ落下
  spawn-fruit.ts            # フルーツ出現
  get-fruit-config.ts       # フルーツ設定（絵文字・得点・出現条件）
  get-available-fruits.ts   # スコアに応じた出現可能フルーツ一覧
  pick-fruit-kind.ts        # フルーツ種類のランダム選択
  check-catch.ts            # キャッチ判定
  check-miss.ts             # ミス判定
  game-loop.ts              # ゲームループ（setInterval）
  render-frame.ts           # ゲーム画面描画
  render-title.ts           # タイトル画面描画
  render-gameover.ts        # ゲームオーバー画面描画
  render-centered-screen.ts # 中央揃え画面の共通描画
  *.test.ts                 # 各関数のユニットテスト
```
